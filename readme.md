### 📋 Pré-requisitos

De que coisas você precisa para instalar o software e como instalá-lo?

* [NodeJs v16.14.2](https://nodejs.org/en/)

* [Mysql](https://www.mysql.com/downloads/)

```
$ npm install -g nodemon
```
```
$ npm install -g ts-node
```
